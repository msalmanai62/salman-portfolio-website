Weather Station and Temperature Controlled AC

An IOT and AI Project for Live and Future Weather Forecasting based on Machine Learning Techniques. It Uses Arduino, Temperature, humidity, pressure and rain sensors to collect data from the envrionment and sends live data to MySQL server.
We stored data in MySQL database and then applied random forest machine learning algorithms to predict rain and temperature. And based on the temperature we are controlling AC. If temperature falls down certain limit it turns off the AC and vice versa. Tools and methods used in this project are aurduino, sensors, python, database, c++, mySQL, and machine learning algorithms.
